Hexo+Github搭建blog

2018-05-21
Blog, Hexo

Hexo是一个快速、简洁且高效的博客框架。Hexo使用Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。同时HEXO还有很多美观的主题可以随意选择，配置起来也是简单容易上手。

工具

  • Git
  • Node.js
  • HEXO
  • 一个域名
  • 免费DNS服务器
  • Mac环境

5、绑定自己的域名

这个过程前前后后持续了一个星期，因为实名认证用了很长时间，再加上配置DNS服务器也需要实现生效。
首先买个域名，我从万网7块钱买了一个域名一年的使用期限，找一些冷门一点的后缀价格都比较便宜，万网就是阿里云。但是阿里云的域名需要实名认证，这个时间还挺久的，3-5个工作日。如果不想走实名认证可以买个墙外的，这个有很多，狗爹啥的都行。很简单，就是网购的流程。
之后需要配置DNS服务器。从阿里云后台配置该域名的DNS服务器：f1g1ns1.dnspod.net 和 f1g1ns2.dnspod.net ，配置后如下图：
